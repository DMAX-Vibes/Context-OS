# End of Session — Context OS Update

Paste this at the end of any meaningful session, on any tool. This is the loop that keeps Context OS current instead of letting it rot.

```text
Before we finish — did this session reveal any durable context?

If yes, propose patches to the right files (YOU_CONTEXT.md, YOU_PROJECTS.md,
YOU_TECH_STACK.md, YOU_CAREER.md, YOU_CREATIVE.md, a project-specific
PROJECT_CONTEXT.md/CLAUDE.md, or a *_PRIVATE.md overlay if it's sensitive).

For each: say where it goes and show the exact edit. Don't add ephemeral details,
one-off tasks, or anything sensitive to the default (non-private) files.

If a lesson from one project generalizes, also record the general version in the
right cross-project file (usually YOU_TECH_STACK.md or YOU_CONTEXT.md).

Wait for my approval before writing anything.
```
